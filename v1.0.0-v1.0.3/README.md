# Embarrassment Principle: Oscillating Dark Energy Model

**Evidence for a Rapidly Oscillating Dark Energy Component from the Embarrassment Principle and Pantheon+ BAO Data**

## Overview

This repository contains the code, LaTeX manuscript, and figures for an oscillating dark energy model motivated by the **Embarrassment Principle**:

> *All real physical systems avoid absolute static equilibrium and naturally oscillate around equilibrium.*

We fit the model to **Pantheon+ SNIa** (full covariance matrix) and **BOSS DR12 BAO** data, with **Planck 2018** priors. The model is derived from first principles (the Embarrassment Principle), not from phenomenological data fitting.

## Key Results

### Free fit (α ≥ 0) — main result
| Parameter | Value |
|-----------|-------|
| \(A\) (amplitude) | 0.600 |
| \(\alpha\) (damping) | 0.00 |
| \(\beta\) (frequency) | 22.5 |
| \(\Omega_m\) | 0.299 |
| \(H_0\) (km/s/Mpc) | 72.00 |
| \(\Delta\chi^2\) (vs ΛCDM) | 109.3 |
| **BIC** | **86.9** |
| \(\Delta\text{AIC}\) | 103.3 |

### Conservative fit (α ≥ 0.1) — robustness test
| Parameter | Value |
|-----------|-------|
| \(A\) | 0.600 |
| \(\alpha\) | 0.10 |
| \(\beta\) | 22.5 |
| \(\Omega_m\) | 0.299 |
| \(H_0\) | 72.00 |
| \(\Delta\chi^2\) | 108.5 |
| **BIC** | **86.2** |
| \(\Delta\text{AIC}\) | 102.5 |

A shallow \(\chi^2\) plateau appears for \(\alpha \in [0, 0.1]\), indicating a parameter degeneracy that future high-redshift data (DESI, Euclid, Roman) will break.

## Model

\[
w(z) = -1 + A e^{-\alpha z} \sin(\beta z)
\]
